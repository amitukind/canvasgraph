A Simple HTML5 Canvas 2D Graph Demo with movable points.
1. Graph creation using canvas
2. Points Drawing
3. Points Dragging
4. Snapping to graph lines

![](images/poster.jpg)


## Usage
Run index.html in any local server or use below
```bash
npm install http-server
http-server
```
